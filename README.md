# Kütüphane Yönetim Sistemi

Modern bir kütüphane yönetim sistemi. Supabase ve Next.js ile geliştirilmiştir.

## Özellikler

- 📚 Kitap kataloğu yönetimi
- 👥 Üye yönetimi
- 📅 Ödünç verme işlemleri
- 💰 Ceza hesaplama sistemi
- 🔐 Güvenli kimlik doğrulama
- 📱 Responsive tasarım
